# onta

Onta SDK and CLI. Alias of [`cograph`](https://www.npmjs.com/package/cograph) — same SDK, same CLI, published under the Onta brand. Install with `npm install -g onta` (CLI) or `npm install onta` (SDK: `import { Client } from "onta"`). See the [cograph README](https://github.com/git-moeen/onta-oss/tree/main/packages/cograph) for full command and API docs.
